## Conference Acronyms (Strict)
You must strictly follow these naming conventions for conference acronyms.
- **USENIX Security** -> `usenix` (Note: We ONLY collect USENIX Security. Ignore ATC, FAST, etc. unless specified.)
- **OSDI** -> `osdi`
- **NDSS** -> `ndss`
- **IEEE S&P** -> `sp`
- **ACM CCS** -> `ccs`
- **ISSTA** -> `issta`
- **ICSE** -> `icse`
- **CVPR** -> `cvpr`
- **NeurIPS** -> `neurips`
- **ICML** -> `icml`
- **ICLR** -> `iclr`
- *Others*: Use the most common lowercase academic acronym.
